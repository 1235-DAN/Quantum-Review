核心概念與古典 vs 量子比較

✕ 古典物理的失敗

  • 能量傳遞是連續的，可取任意值
  • 光是電磁波，強度由振幅決定
  • 光照夠久夠強就能打出電子（任何頻率）
  • 短波輻射強度趨近無窮大（紫外災變）
  • 粒子就是粒子，沒有波動性

✓ 量子物理的突破

  • 能量不連續，只能是最小單位的整數倍
  • 光是光子（粒子），強度 = 光子數多寡
  • 光電效應只取決於頻率，與強度無關
  • 能量量子化後完美符合黑體輻射實驗
  • 粒子也有波長（德布羅意物質波）

02

電子的發現：湯姆森荷質比實驗

🔬 實驗背景與意義 19 世紀末，科學家在低壓密閉玻璃管中接上高電壓，發現從陰極射出的神秘射線（陰極射線）能使螢光幕發光、穿越障礙物後留下陰影，且會使小滾輪滾動，顯示其具有動量。 1897 年，湯姆森設計速度選擇器，首次準確測量陰極射線粒子的「荷質比」（e/m），確認它是一種帶負電的微粒——即後來所稱的電子。 這個實驗推翻了原子不可分割的舊觀念，是近代物理的重要里程碑，湯姆森也因此獲得 1906 年諾貝爾物理獎。
⚡實驗裝置示意圖

速度選擇器：電場力（向上）與磁力（向下）平衡，陰極射線不偏折直線前進

📐公式推導

① 速度選擇器（求 v₀）

1電場力（向上）：F_E = eE
2磁力（向下）：F_B = ev₀B
3平衡條件 F_E = F_B：eE = ev₀B
4消去 e 得：v₀ = E / B
速度選擇器v₀ = E / B

② 偏移量計算荷質比（求 e/m）

1關閉磁場，只留電場，電子作拋物線運動
2x 方向等速：t = d / v₀（d = 板寬）
3y 方向加速：a_y = eE/m，y₁ = ½a_y t²
4離開電場後等速：y₂ = v_y · (D/v₀)
5總偏移 y = y₁+y₂，整理得：
荷質比e/m = 2v₀²y / [Ed(d+2D)]
公認值e/m = 1.76 × 10¹¹ C/kg
03

密立坎油滴實驗

🔬 實驗背景與意義 湯姆森測出了電子的荷質比，但無法單獨確定電荷量與質量。美國科學家密立坎從 1909 年起，花近十年時間進行油滴實驗，精確測定電子的電荷量。 實驗方法是讓帶電小油滴在平行板電場中懸浮或以不同速度運動，透過測量終端速度推算油滴電荷，反覆測量不同油滴，發現所有電荷量皆為同一最小值的整數倍。 這個最小值即為基本電量 e，確立了「電荷量子化」的概念，密立坎因此獲得 1923 年諾貝爾物理獎。
🔋實驗裝置與受力分析

無電場：終端速度 v（重力 = 阻力）；有電場：新終端速度 v'（qE = mg + f'）

📐公式推導與實驗結果

① 無電場（測油滴重量）

1斷路 S，油滴在重力下加速，最終達終端速度 v
2終端條件：mg = f（空氣阻力）
3由斯托克定律：f = 6πηrv（η：黏度，r：半徑）

② 有電場（求電荷量）

1接通電路，油滴向上加速，達新終端速度 v'
2新終端條件：qE = mg + f'
3聯立兩式消去 mg，可求出電荷量 q
4反覆測量不同油滴，發現 q 皆為 e 的整數倍
基本電量e = 1.602 × 10⁻¹⁹ C ≈ 1.60 × 10⁻¹⁹ C
電子質量（由荷質比推算）m = e ÷ (e/m) ≈ 9.11 × 10⁻³¹ kg
⚡ 電荷量子化 任何物體的電荷量必為 e 的整數倍（ne），電荷不可無限細分。這是量子化概念在電量上的首次確認。
04

X 射線：發現、產生與布拉格繞射

🔬 實驗背景與意義 1895 年，德國物理學家倫琴（命琴）在研究陰極射線時，意外發現一種新型輻射能穿透紙張和肌肉，使感光底片曝光——他將其命名為「X 射線」，因其性質當時尚屬未知。 1912 年，勞厄以 X 射線照射晶體，得到清晰繞射圖案，證明 X 射線是波長極短（0.01～10 nm）的電磁波，並可用於分析晶體原子結構。 布拉格父子進一步建立定量繞射公式（布拉格定律），兩人因此共同獲得 1915 年諾貝爾物理獎，X 射線晶體學自此成為材料科學的重要工具。
⚡X 射線的產生

電子動能（eV）在撞靶瞬間轉換為電磁波能量，產生波長極短的 X 射線

1

加速電子

高電壓（萬伏特）將陰極電子加速，轉換電位能為動能：eV = ½mv²

2

撞擊金屬靶急遽減速

電子撞擊正極靶材，在極短時間內急遽減速，損失的動能以電磁波形式釋出

3

放出 X 射線

所產生的電磁波波長極短（0.01～10 nm），即 X 射線；不帶電，不受電磁場偏折

🔬布拉格晶體繞射

θ 為掠射角（射線與晶面的夾角），非入射角！兩條光路的光程差為 2d sinθ

布拉格定律2d sinθ = nλ　（n = 1, 2, 3…）
⚠ 布拉格定律推導邏輯 第二層原子的 X 射線需多走 2d sinθ 的路程。若此光程差恰好等於波長的整數倍（nλ），兩道反射光的波峰重合，發生建設性干涉（亮紋）；否則相消。
🌐 X 射線的實際應用
🦷

醫學影像（X 光片 / CT）

X 射線能穿透肌肉但被骨骼吸收，各組織吸收率不同，可在底片上產生明暗對比。現代 CT（電腦斷層）整合不同角度的 X 光片，重建三維立體影像，大幅提升診斷精度。

🛄

機場安全檢查

機場行李掃描機使用兩束能量不同的 X 射線，辨識不同材質：有機物呈橘色、金屬呈藍綠色、不可穿透物呈黑色，無需開箱即可識別危險物品。

🔬

晶體結構分析（X 射線繞射）

利用布拉格定律，由繞射角推算晶格間距，進而確定晶體原子排列。DNA 雙螺旋結構的發現正是依賴 X 射線繞射圖，是 20 世紀最重要的科學成就之一。

🏭

工業非破壞性檢測（NDT）

X 射線可穿透金屬焊接件，偵測內部裂縫、氣泡等缺陷，而不破壞工件本身。廣泛應用於航空零件、橋樑結構、核電廠設備的品質管控。

⚗️

X 射線螢光分析（XRF）

X 射線激發樣品中原子發出特徵螢光 X 射線，由光譜判別元素組成，可快速分析礦石、文物、土壤的元素含量，且通常不損傷樣品。

☀️

天文觀測（X 射線望遠鏡）

高能天體（黑洞、中子星、超新星殘骸）會放出大量 X 射線。由於地球大氣吸收 X 射線，需用太空望遠鏡（如錢卓 X 射線天文台）在大氣層外觀測這些極端天體。

黑體輻射、空腔輻射與普朗克量子論

🔬 背景：為什麼需要黑體輻射理論？ 任何溫度高於 0 K 的物體都會放出電磁波（熱輻射），但其輻射強度分布的形狀，依材質、表面狀態的不同而各異，難以建立統一理論。 科學家為了排除材質影響，設想一種「理想吸收體」——黑體：能完全吸收所有外來輻射，其輻射強度分布純粹由溫度決定。 古典物理嘗試解釋黑體輻射曲線，在長波段尚可，但在短波段預測輻射強度趨近無窮大，與實驗嚴重矛盾，這個困境稱為「紫外災變」，最終由普朗克在 1900 年引入量子化概念解決，開創了量子力學。
📦黑體與空腔輻射

有小孔的空腔可視為理想黑體：進入的輻射幾乎不可能反射出來，能量被完全吸收

1

什麼是黑體？

能完全吸收外來電磁輻射（吸收率 = 1），不反射任何光；不一定是黑色（如恆星即被視為黑體）

2

什麼是空腔輻射？

有小孔的空腔達熱平衡時，由小孔放出的電磁輻射即為黑體輻射，其能量分布僅與溫度有關，與腔壁材質無關

3

為何恆星視為黑體？

恆星表面放出的光幾乎全是自身熱輻射，反射外來光的比例極小，因此在天文學中視恆星為黑體，可由輻射峰值波長推算表面溫度

📊黑體輻射曲線與紫外災變

實線為量子理論預測（吻合實驗），紫色虛線為古典預測（紫外災變）

維因位移定律λ_max · T = b　（b ≈ 2.9×10⁶ nm·K）
⚛普朗克量子假設與能量量子化

普朗克假設空腔壁上的振子（電偶極），吸收或放出的能量不是連續的， 只能是某一最小能量單位（能量量子 ε）的整數倍。 這個假設違背了古典物理「能量連續」的觀念，但完美解釋了黑體輻射曲線。

能量量子ε = hf
振子能量（量子化）E = nhf　（n = 1, 2, 3…）
普朗克常數h = 6.626 × 10⁻³⁴ J·s ≈ 6.63 × 10⁻³⁴ J·s
📅 量子論的誕生日 1900 年 12 月 14 日，普朗克在德國物理學會公開報告量子化假設。 這一天被稱為量子論的誕生日，也是近代物理與古典物理的分界。普朗克因此獲得 1918 年諾貝爾物理獎。

古典物理允許任意能量值（斜面）；量子化後能量只能在離散能階上（階梯）

雷納的光電效應實驗

🔬 實驗背景與意義 1887 年赫茲首先觀察到紫外線照射電極可使放電更容易，後來的學者認為這正是光電效應的首次發現。 1902 年，德國科學家雷納（Philipp Lenard）設計精密的光電管實驗，系統化研究了光電效應，測量了光電流與電壓的關係，發現了「底限頻率」、「截止電壓」等重要現象，揭示出古典電磁波理論無法解釋的矛盾。 雷納的實驗為愛因斯坦 1905 年提出光量子論提供了最直接的實驗依據，雷納本人也因此獲得 1905 年諾貝爾物理獎。
💡實驗裝置與電流電壓關係

光照射金屬靶，打出光電子形成光電流 I；可調電壓控制順向（加速）或逆向（阻止）

⬆ 順向電壓（正電壓）

收集極接正極，對光電子有加速作用，協助更多電子到達收集極。電壓增大到某值時，幾乎所有光電子都被收集，光電流達最大值（飽和電流）。

⬇ 逆向電壓（負電壓）

收集極接負極，對光電子有阻擋作用，只有動能夠大的電子才能到達。電壓增大到截止電壓 Vc 時，即使最快的電子也無法到達，光電流 = 0。

📈I–V 圖與三大發現

相同頻率不同強度：截止電壓 Vc 相同（截距不變），飽和電流不同（水平段高度不同）

愛因斯坦光量子論

🔬 理論背景與意義 1905 年，愛因斯坦受普朗克量子論啟發，大膽假設：光在傳播過程中本身也具有粒子性，不只在被吸收或放出時才量子化。他認為光是由一個個「光量子」（1926 年改稱光子）組成，每個光子攜帶固定能量 hf。 這個假設完美解釋了雷納實驗的所有奇異現象——底限頻率的存在、最大動能只與頻率有關、幾乎立即產生光電流等，但因為與「光是連續電磁波」的觀念衝突，初期並未被廣泛接受。 密立坎耗費近十年實驗驗證，最終證實了愛因斯坦的光電方程式；愛因斯坦因「發現光電效應定律」獲得 1921 年諾貝爾物理獎，而非因相對論，光子論的重要性由此可見一斑。
☀光子的基本性質

光子同時具有波動性（波長、頻率）和粒子性（能量、動量）——波粒二象性

光子能量E = hf = hc / λ
光子動量（愛因斯坦由狹義相對論推導）p = E/c = hf/c = h/λ
💡 光強度的量子解釋 古典物理：強度 ∝ 振幅²（電場能量）
量子理論：強度 = 單位時間單位面積的光子數 × hf
即 I = nhf/A（n：每秒每單位面積通過的光子數）
📐光電方程式推導與 Vc–f 圖

愛因斯坦光電方程式推導

1光子能量為 hf，撞擊金屬表面電子
2電子用部分能量克服功函數 W（脫離束縛）
3剩餘能量成為光電子動能：hf = W + K_max
4截止電壓與最大動能的關係：K_max = eVc
5功函數與底限頻率：W = hf₀ = hc/λ₀
光電效應方程式K_max = hf − W = eVc

密立坎光電實驗：Vc 對 f 作圖，各金屬直線斜率相同均為 h/e，驗證愛因斯坦方程式

📌 與 Vc–f 圖的關係 Kmax–f 圖其實就是把 Vc–f 圖的縱軸乘以 e（因為 Kmax = eVc）。 兩張圖形狀完全相同（平行斜線、相同 x 截距），但斜率和 y 截距的「數值」不同： 斜率從 h/e 變成 h，y 截距從 −W/e 變成 −W，讀圖時要特別注意。

方程式推導（Kmax–f 直線的由來）

1光電方程式本身就是直線方程：K_max = hf − W
2對應 y = mx + b：
3斜率 m = h（普朗克常數，各金屬相同）
4y 截距 b = −W（功函數負值，各金屬不同）
5令 K_max = 0：f = f₀ = W/h（x 截距）

🎯 看圖解題四步驟
① 求 h：取兩點算斜率。Vc–f：h = 斜率 × e；K–f：h = 斜率
② 求 f₀：讀 x 截距即為底限頻率，兩圖相同
③ 求 W：Vc–f：W = |y截距| × e；K–f：W = |y截距|（直接讀出）
④ 比較金屬：斜率相同（h 不變）；x 截距大 → f₀ 大 → W 大（功函數大）
✅ 光子論如何解釋雷納的三大發現
現象古典困境光子論的解釋
底限頻率 古典：能量可累積，任何頻率照夠久都行 一個光子能量 hf 必須 ≥ W 才能打出電子。頻率 f < f₀ 時，hf < W，單個光子能量不足，無論多少光子都無法打出電子（能量不能累積）
幾乎立刻產生 古典：能量需時間累積，應有延遲 光子與電子是一對一碰撞，只要光子能量 ≥ W，碰撞後立即給予電子能量，幾乎無延遲（< 10⁻⁹ 秒）
最大動能與強度無關 古典：光越強，電場越大，電子動能應越大 每個光子能量 = hf（只取決於頻率）。增加強度只是增加光子數，每個電子仍只吸收一個光子，動能仍為 hf - W
飽和電流 ∝ 光強度 古典：部分可解釋 光強度越大 → 光子數越多 → 每秒打出的電子數越多 → 飽和電流越大。這部分古典與量子理論預測相同

⚖基本單位換算

能量單位換算關係：J（標準）⟷ eV（原子物理常用）；波長單位 Å 和 nm

能量換算（最常用）1 eV = 1.6 × 10⁻¹⁹ J
hc 快速換算值hc ≈ 12400 eV·Å = 1240 eV·nm
🔢光子能量三種計算方式

方法一：用頻率 f 計算（標準 SI 單位）

1已知頻率 f（Hz），代入 E = hf
2h = 6.63×10⁻³⁴ J·s，結果單位為 J
3若需 eV：÷ 1.6×10⁻¹⁹

方法二：用波長 λ（m）計算

1已知 λ（m），代入 E = hc/λ
2hc = 6.63×10⁻³⁴ × 3×10⁸ ≈ 1.99×10⁻²⁵ J·m

方法三：用波長 λ（Å）快速算（考試推薦！）

1將 λ 換算為 Å：1 nm = 10 Å，1 m = 10¹⁰ Å
2直接用 E(eV) = 12400 / λ(Å)
3例：λ = 5000 Å → E = 12400/5000 = 2.48 eV
🚀 考試速算技巧 記住 E(eV) = 12400 / λ(Å)，這個公式幾乎涵蓋所有光電效應計算題。 換算關係：可見光 400–700 nm = 4000–7000 Å，能量 1.77–3.10 eV。

⚠ 考試最常見的陷阱（每年都考！）
陷阱 1：光強度 ≠ 截止電壓增加光強度 → 光子數↑ → 飽和電流↑
但截止電壓 Vc 和最大動能完全不變！
陷阱 2：布拉格定律的 θθ 是掠射角（射線與晶面夾角），
不是射線與法線夾角（≠ 光學入射角）
陷阱 3：eV 與 V 的混淆截止電壓 Vc 單位是 V（伏特），
K_max = eVc，eVc 的單位才是 J；但數值上 K_max = Vc（eV）
陷阱 4：底限頻率 vs 底限波長頻率越高 → 能量越大 → 底限頻率是最小頻率
但底限波長 λ₀ 是對應的最長波長
